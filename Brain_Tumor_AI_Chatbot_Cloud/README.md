# Brain Tumor AI Chatbot — Cloud Deploy

Ready to deploy on Streamlit Cloud, Hugging Face Spaces, or Render.

## Deploy on Streamlit Cloud
1. Push this folder to GitHub.
2. Go to https://share.streamlit.io → New App.
3. Select repo, set `app.py` as main file, deploy.
4. (Optional) Add `MODEL_URL` as an environment variable for auto-download.

## Requirements
- TensorFlow CPU 2.17.0 (works on Python 3.11)
- Streamlit 1.39.0
